# 软著申请 - 填表说明

## 一、中国版权保护中心在线填报地址

**官方网站：** https://www.ccopyright.com.cn/

**填报入口：** 计算机软件著作权登记

## 二、软件著作权登记申请表填写内容

### 2.1 软件基本信息

| 字段 | 填写内容 | 说明 |
|------|----------|------|
| **软件名称** | 薪酬助手系统 | 建议不超过30字 |
| **软件简称** | 薪酬助手 | 可选 |
| **版本号** | V1.0 | 当前版本 |
| **开发完成日期** | 2025-01-01 | 填实际开发完成日期 |
| **发表状态** | 已发表/未发表 | 根据实际情况选择 |
| **发表日期** | 留空或填实际日期 | 如已发表 |
| **软件类型** | 应用软件 | 业务系统 |
| **行业分类** | 企业管理软件 | 薪酬/HR领域 |
| **开发方式** | 独立开发 | - |
| **著作权人** | [公司名称/个人姓名] | 申请人 |
| **权利范围** | 全部权利 | 默认 |

## 四、权利归属证明材料（如需）

- 企业开发：营业执照复印件
- 个人开发：身份证正反面复印件
- 委托开发：委托开发合同 + 权利归属协议
- 合作开发：合作协议 + 权利归属协议

## 五、在线填报流程

1. **注册账号** → 中国版权保护中心官网
2. **实名认证** → 企业/个人认证
3. **在线填报** → 填写软件基本信息
4. **上传材料** → 源程序 + 软件说明书
5. **在线缴费** → 一般300元/件
6. **受理审查** → 30-60个工作日
7. **取得证书** → 电子证书/纸质证书
